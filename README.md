# 5-Day Weather Forecast
A simple application to display 5-day weather forecast using the OpenWeatherMap API.

## Pre-requisites 
* Node.js 6.1 and above

## How to run
* Clone it
* ```cd react-weather-forecast```
* ```npm i```
* ```npm start```
* Open ```http://localhost:8080``` to see the app.

### Tech Stack

* React.js
* Redux
* ES6g
